# Sidelink Docs

Sidelink is a self-hosted iOS sideload manager with a cross-platform TypeScript signing engine, desktop app, web control center, and iOS helper companion.

## Product Principles

- Local-first operation with no third-party relay
- Explicit security boundaries and least-privilege defaults
- Production-minded UX for onboarding and maintenance
